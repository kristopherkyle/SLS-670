Tasks for the day:

For each of the three variables (Score, Nwords, Frequency) in the .csv file “distribution_sample.csv” do the following:

  • Find the mean, median, and standard deviation
  • Create a histogram to visually check for normality
  • Create a qqplot to visually check for normality
  • Run a Shapiro-Wilk’s test to check for normality

Don’t forget to load ggplot2 and the data!

library(ggplot2) #load ggplot2
#load data - (don't forget to set your working directory!)
sampledata <- read.csv("data/distribution_sample.csv", 
                       header = TRUE)

1. Find the mean, median, and standard deviation

mean(sampledata$Score)
## [1] 3.427083
median(sampledata$Score)
## [1] 3.5
sd(sampledata$Score)
## [1] 0.8871007

2. Create a histogram

This is the code used to make the previous plot:

ggplot(sampledata, aes(x = Score)) +
  geom_histogram((aes(y=..density..)), color = "black")

3. Create qq plot

This is the code used to make the previous plot:

ggplot(sampledata, aes(sample = Score)) +
  stat_qq() +
  stat_qq_line()

4. Run the Shapiro-Wilk’s test

## 
##  Shapiro-Wilk normality test
## 
## data:  sampledata$Score
## W = 0.95555, p-value = 7.671e-11

This is how we run the Shapiro-Wilk’s test:

shapiro.test(sampledata$Score)
## 
##  Shapiro-Wilk normality test
## 
## data:  sampledata$Score
## W = 0.95555, p-value = 7.671e-11

According to this test, is Score normally distributed?
